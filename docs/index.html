Core Positioning

Local memory that pays for itself

openclaw plugin install https://github.com/philonis/mem8/blob/main/SKILL.md

四个核心价值，一眼看懂

仿照 mem9 的“基础设施感”，但把 mem8 的独特点拉出来：本地优先、无需运维、可直接省 Token。

01

本地持久记忆

用 SQLite 做默认后端，跨会话、跨重启保留记忆，不依赖任何云数据库或远程服务。

02

Token 节省可量化

不是“多一个记忆层”而已。通过结构化召回和记忆压缩，离线 benchmark 已看到约 36% 的 Token 降幅。

03

零额外运维

没有向量数据库、没有同步脚本、没有远程控制面板。安装完成就能写入、召回和清理记忆。

04

自动 hygiene

内置相似记忆合并、陈旧噪音清理、compact 优先保留高价值记忆，避免“越用越脏”。

mem8 的优势不止是“记忆”

如果 mem9 解决的是 memory persistence，那么 mem8 解决的是 memory persistence + prompt efficiency。

安装方式尽量做到“一句话讲清”

最好是一个 SKILL.md URL，直接丢给 OpenClaw 就能装，这也是页面里最该醒目的地方。

1

OpenClaw 一键安装

直接把下面这行发给 OpenClaw，即可从 GitHub 安装插件。

openclaw plugin install https://github.com/philonis/mem8/blob/main/SKILL.md
2

手动安装

如果你想自己控制目录和依赖，也可以直接克隆。

git clone https://github.com/philonis/mem8.git ~/.openclaw/plugins/mem8 cd ~/.openclaw/plugins/mem8 && npm install
3

默认配置即可启动

默认使用 SQLite 和 Ollama，本地目录不存在时会自动创建。首次上线几乎不用额外接线。

{ "embeddingProvider": "ollama", "embeddingModel": "nomic-embed-text:latest", "embeddingUrl": "http://127.0.0.1:11434", "dbPath": "~/.mem8/memories.sqlite" }

English Version

mem8 is a local-first memory layer for OpenClaw. It persists memory with SQLite, performs local semantic recall with Ollama, and reduces context cost while improving long-session consistency.

  • Local-first by default, no cloud dependency
  • Persistent memory across restarts and sessions
  • Roughly 36% token savings in offline benchmarks
  • Installable through a single SKILL.md URL
